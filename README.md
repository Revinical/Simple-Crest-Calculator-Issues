Get the latest release from one of these sources:

- **CurseForge:** [[SimpleCrestCalculator on CurseForge](https://www.curseforge.com/wow/addons/simple-crest-calculator)]

---

## 🐛 Reporting a Bug

Before opening an issue, please:

1. Make sure you're on the **latest version** of the addon
2. Check [existing issues](../../issues) to avoid duplicates
3. Disable other addons temporarily to rule out conflicts

When you open a bug report, please include:
- **Your WoW version** (Retail / Classic Era / Cata Classic)
- **Addon version** (shown in the addon list in-game)
- **What happened** vs **what you expected**
- Any relevant **error messages** (the full text, not a screenshot if possible)

---

## 💡 Suggesting a Feature

Feature requests are welcome! Please describe:
- **What** you'd like the addon to do
- **Why** it would be useful
- Any examples from other addons you've seen do something similar

## ❓ FAQ

**Can I contribute code?**  
The source code is maintained in a private repository. Code contributions aren't 
accepted at this time, but feedback and ideas absolutely are.

**How do I know if my issue was fixed?**  
Issues will be closed with a comment referencing the version the fix ships in. 
Keep an eye on CurseForge/Wago for release notes.

**Where are the patch notes?**  
Release notes are posted on the CurseForge and Wago addon pages with each update.
